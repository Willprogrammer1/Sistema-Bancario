# Sistema Bancário Web

Projeto pessoal desenvolvido com HTML, CSS e JavaScript, simulando um sistema bancário simples.

## Funcionalidades
- Adicionar saldo
- Remover saldo
- Validação de saldo insuficiente
- Interface moderna estilo aplicativo
- Modal interno (sem alert do navegador)
- Aplicação instalável (PWA)

## Tecnologias
- HTML5
- CSS3
- JavaScript
